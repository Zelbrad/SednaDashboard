A modern, high-performance crypto dashboard built with React, Vite, and Tailwind CSS. Features real-time market data visualization, asset tracking, and a premium "glassmorphism" UI design.

## Features
*   Real-time crypto market data
*   Interactive charts and metrics
*   Responsive design with mobile support
*   Premium dark mode UI

## Run Locally

1.  Install dependencies:
    ```bash
    npm install
    ```
2.  Run the development server:
    ```bash
    npm run dev
    ```
